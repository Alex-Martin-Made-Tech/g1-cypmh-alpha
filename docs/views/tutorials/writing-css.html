Writing CSS

CSS used in the NHS.UK prototype kit is written in the SCSS syntax of Sass.

Sass

Sass is an extension of CSS that gives a load of extra features useful in producing CSS for the mass of different devices and browsers.

You can paste CSS into it without breaking it which is useful for prototyping.

Writing code

You write your Sass in app/assets/sass and it gets compiled into the CSS used in your page (found in /public/css).

There is already a CSS file included to use called main.scss which compiles into main.min.css.

Note that Sass files are identified by the .scss extension.

Every time a change happens in main.min.scss it will produce a new version of main.min.css. Make sure to write your css in main.scss as anything you put in main.min.css will get overridden.
