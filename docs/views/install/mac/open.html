Terminal basics

The terminal lets you type in commands and run programs on your computer.

You can also use it to browse your file system - to open folders, etc.

Learn the terminal basics

You do not need to do this now, but learning a few basic terminal commands can make using the kit much easier.

Go to your prototype

You need to navigate to your prototype in the terminal. Most commands for the kit need to be run in the prototype folder.
