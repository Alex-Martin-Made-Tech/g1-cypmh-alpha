Macs come with Terminal.app. It’s located in the Utilities folder in the Applications folder.

You can also find it using spotlight (magnifying glass icon in the top right) and typing 'terminal'.

Entering the commands in the terminal

Commands to be entered in to the terminal will be inset like this:

command to be typed in to terminal

Once you’ve typed the command, press enter to send it.
