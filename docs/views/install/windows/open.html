Git bash basics

Git bash lets you type in commands and run programs on your computer.

You can also use it to browse your file system - to open folders, etc.

Learning a few basic git bash commands can make using the kit much easier.

  • Tutorial on using terminal (mac)
  • Tutorial on using git bash (windows)

Go to your prototype

You need to navigate to your prototype in the git bash. Most commands for the kit need to be run in the prototype folder.

cd ~/Documents/projects/juggling-licence-prototype

Please note: Make sure your project has the same name and is in the same folder as in step 7.
