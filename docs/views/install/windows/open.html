Git bash basics

Git bash lets you type in commands and run programs on your computer.

You can also use it to browse your file system - to open folders, etc.

Learn git bash basics

You do not need to do this now, but learning a few basic git bash commands can make using the kit much easier.

Go to your prototype

You need to navigate to your prototype in the git bash. Most commands for the kit need to be run in the prototype folder.
